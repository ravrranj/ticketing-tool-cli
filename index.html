Ticketing Tool CLI

A simple Java command-line interface tool for managing support tickets with priority and status updates.

Features

  • Add priority and normal tickets
  • View ticket queues
  • Process tickets based on priority
  • Update ticket statuses

How to Use

1. Run the Java program
2. Use the numbered menu options:
   - Add Ticket
   - View Queue
   - Process Next Ticket
   - Update ticket status
   - Exit

Getting Started

Clone this repository and run the main Java file:

git clone https://github.com/ravrranj/ticketing-tool-cli.git
cd ticketing-tool-cli
javac Main.java
java Main
